# EDITH

Sonnet 4.6-backed daily-driver agent. Surface: `edith.chat`.

EDITH is a Nexus-style chat node — point a client at `edith.chat`, send
`{message, conversation_id?}`, get back `{reply}`. Conversation history is
held in-process, keyed by `conversation_id` (default `"default"`), and capped
at 20 turns per conversation.

## Boot

Requires `hosts/mac-mini/.env` to exist (generated by `scripts/bootstrap.sh`)
with `EDITH_SECRET` and `ANTHROPIC_API_KEY`. RAVEN_MESH Core must already
be running on the host at `http://host.docker.internal:8000`.

```bash
cd nodes/edith
docker compose up --build
```

The container connects out to Core via `host.docker.internal:8000` (mapped
to the Mac mini host).

## Verify it registered

From any host on the Tailnet:

```bash
curl http://100.109.10.50:8000/v0/healthz
# expect: nodes_connected >= 1
```

## Test from a client

Use the `human.py` pattern from `~/raven/mesh_demo/`. Minimal example —
once you have a `human` node registered with an edge into `edith.chat`:

```bash
HUMAN_SECRET=<hex> CORE_URL=http://100.109.10.50:8000 python3 - <<'PY'
import asyncio, datetime as dt, hashlib, hmac, json, os, uuid, aiohttp

SECRET = os.environ["HUMAN_SECRET"].encode()
def sign(env):
    body = {k: v for k, v in env.items() if k != "signature"}
    return hmac.new(SECRET, json.dumps(body, sort_keys=True, separators=(",", ":")).encode(), hashlib.sha256).hexdigest()
def now(): return dt.datetime.now(dt.timezone.utc).strftime("%Y-%m-%dT%H:%M:%SZ")

async def go():
    async with aiohttp.ClientSession() as s:
        reg = {"node_id": "human", "timestamp": now()}; reg["signature"] = sign(reg)
        await s.post("http://100.109.10.50:8000/v0/register", json=reg)
        env = {"id": str(uuid.uuid4()), "from": "human", "to": "edith.chat",
               "kind": "invocation", "payload": {"message": "hello edith"},
               "timestamp": now()}
        env["correlation_id"] = env["id"]; env["signature"] = sign(env)
        async with s.post("http://100.109.10.50:8000/v0/invoke", json=env) as r:
            print(r.status, await r.json())
asyncio.run(go())
PY
```

## Logs

```bash
docker compose logs -f edith
```

All log lines are prefixed `[edith]`.

## Model

The Claude model id is set via the `EDITH_MODEL` env var (default
`claude-sonnet-4-6`). To use a different Sonnet revision, override it in
`hosts/mac-mini/.env`:

```
EDITH_MODEL=claude-sonnet-4-5-20250929
```
